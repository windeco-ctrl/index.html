Hướng dẫn cài đặt Google Sheets
1

Tạo Google Sheet mới

Vào sheets.google.com → tạo spreadsheet tên TikTok Dashboard

2

Tạo 3 tab

Tab Channels:

id | name | handle | category | color

Tab Videos:

channel_id | title | date | duration | views | avg_watch | likes | comments | shares | completion | note

Tab Metrics (tuỳ chọn, lưu theo kỳ):

channel_id | date | followers | views | likes | comments | shares
3

Lấy Sheets ID

URL: docs.google.com/spreadsheets/d/ID_Ở_ĐÂY/edit

4

Tạo API Key

  • Vào console.cloud.google.com
  • Bật Google Sheets API → Credentials → tạo API Key
5

Chia sẻ Sheet

Share → "Anyone with the link can view"
