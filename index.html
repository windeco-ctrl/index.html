Hướng dẫn cài đặt Google Sheets (5 bước)
1

Tạo Google Sheet mới

Mở sheets.google.com → tạo spreadsheet mới. Đặt tên: TikTok Dashboard

2

Tạo 2 tab theo cấu trúc này

Tab 1: Channels — thêm các cột:

id | name | handle | category | color

Tab 2: Data — thêm các cột:

channel_id | period | followers | follower_delta | views_avg | videos | likes | comments | shares | watchtime | note
3

Lấy Sheets ID

Nhìn vào URL của Google Sheet:
https://docs.google.com/spreadsheets/d/ĐÂY_LÀ_SHEETS_ID/edit

4

Tạo Google Sheets API Key

  • Vào console.cloud.google.com
  • Tạo project mới → bật Google Sheets API
  • Vào Credentials → tạo API Key
  • Copy API Key dán vào ô bên trên
5

Chia sẻ Sheet (quan trọng)

Trong Google Sheet → Share → chọn "Anyone with the link can view"

Deploy lên web (Vercel — miễn phí)
1

Upload file lên GitHub

  • Vào github.com → đăng ký/đăng nhập
  • Tạo repository mới, tên: tiktok-dashboard
  • Upload file index.html vào repo
2

Deploy lên Vercel

  • Vào vercel.com → đăng nhập bằng GitHub
  • Bấm New Project → chọn repo tiktok-dashboard
  • Bấm Deploy — xong! Web sẽ có địa chỉ kiểu tiktok-dashboard.vercel.app
3

Cập nhật sau này

Chỉ cần upload file index.html mới lên GitHub — Vercel tự deploy trong 30 giây.
